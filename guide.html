音声文字起こしツール 使い方ガイド

🚀 クイックスタート

  1. 録音ボタンをクリック - 中央の大きな青いボタンを押します
  2. マイクの許可 - ブラウザがマイクへのアクセスを求めたら「許可」をクリック
  3. 話し始める - 音声がリアルタイムでテキストに変換されます
  4. 録音停止 - 赤い停止ボタンをクリックして終了
💡 ヒント: スペースキーを押すだけで録音の開始/停止ができます！

📋 基本機能

録音と文字起こし

  • リアルタイムで音声をテキストに変換
  • 日本語に最適化された高精度な認識
  • 録音時間と状態の表示
  • ノイズレベルの監視と警告

テキスト編集

  • 録音停止後、テキストを直接編集可能
  • 検索・置換機能（Ctrl + F）
  • 元に戻す/やり直し（Ctrl + Z / Ctrl + Y）
⚠️ 注意: 録音は約60秒で自動的に一時停止しますが、自動的に再開されます。

⌨️ キーボードショートカット

Space 録音開始/停止
Ctrl + F 検索・置換
Ctrl + Z 元に戻す
Ctrl + Y やり直す
Ctrl + S TXTファイルとして保存
Ctrl + N 新規セッション
? ヘルプ表示

⚙️ 設定オプション

ダークモード

目に優しい暗いテーマに切り替えできます。夜間の使用に最適です。

文字サイズ調整

14px〜24pxの範囲で文字サイズを調整できます。

句読点の自動挿入

「です」「ます」などの文末に自動的に句読点を追加します。

タイムスタンプ付きエクスポート

エクスポート時に各発言の時間情報を含めることができます。

🔧 トラブルシューティング

マイクが認識されない

  • ブラウザの設定でマイクへのアクセスを許可してください
  • 他のアプリがマイクを使用していないか確認してください
  • ブラウザを再起動してみてください

音声認識が正しく動作しない

  • 静かな環境で使用してください
  • マイクに向かってはっきりと話してください
  • Chrome、Edge、またはSafariを使用してください（Firefoxは非対応）

文字起こしが途中で止まる

  • 約60秒で自動的に再開されます
  • インターネット接続を確認してください
  • 長時間の録音の場合は定期的に保存してください

💡 使用上のヒント

精度を上げるコツ

  • 静かな環境で録音する
  • マイクから20〜30cm離れて話す
  • ゆっくりはっきりと発音する
  • 専門用語は辞書登録機能を活用する

効率的な使い方

  • キーボードショートカットを活用する
  • 長い録音は複数のセッションに分ける
  • 定期的にエクスポートして保存する
  • 編集は録音停止後に行う

🌐 対応ブラウザ

  • ✅ Google Chrome（推奨）
  • ✅ Microsoft Edge
  • ✅ Safari（macOS/iOS）
  • ✅ Opera
  • ❌ Firefox（音声認識非対応）
⚠️ 重要: 音声認識機能を使用するにはインターネット接続が必要です。

問題が解決しない場合は、ブラウザのコンソールでエラーメッセージを確認してください。
